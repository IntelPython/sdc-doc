pandas.Series.str.isalpha¶

Check whether all characters in each string are alphabetic.

This is equivalent to running the Python string method str.isalpha() for each element of the Series/Index. If a string has zero characters, False is returned for that check.

return

Series or Index of bool Series or Index of boolean values with the same length as the original Series/Index.

Limitations¶

Series elements are expected to be Unicode strings. Elements cannot be NaNs.

Examples¶

Check whether all characters in each string are alphabetic¶
import pandas as pd
from numba import njit


@njit
def series_str_isalpha():
    series = pd.Series(['leopard', 'Golden Eagle', 'SNAKE', ''])
    out_series = series.str.isalpha()

    return out_series  # Expect series of True, False, True, False


print(series_str_isalpha())
