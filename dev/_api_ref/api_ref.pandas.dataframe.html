DataFrame¶

This is the main Pandas* data structure representing a table of rows and columns.

DataFrame is a two-dimensional structure with labeled axes. It can be thought of as a dictionary-like container for Series

Constructor¶

DataFrame

Two-dimensional size-mutable, potentially heterogeneous tabular data structure with labeled axes (rows and columns). Arithmetic operations align on both row and column labels. Can be thought of as a dict-like container for Series objects. The primary pandas data structure.
